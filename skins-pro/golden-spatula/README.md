# Skins-Pro Golden Spatula Theme

Copy the folder `skins-pro/golden-spatula/` into the Skins-Pro source repository, then run:

```bash
npm run build
```

This pack includes:
- 20 standard JPG assets
- `strings.json`
- `theme.css`
- compatibility aliases: `background.jpg`, `avatar.jpg`, `decoration.jpg`, `icon-climate.jpg`, `icon-garden.jpg`

The aliases are included because the upstream Skins-Pro README describes `avatar.jpg`, `background.jpg`, and `decoration.jpg` as standard skin files, while this asset set also preserves your requested filenames such as `stage-background.jpg`, `avatar-steve.jpg`, and `decor-wolf-lantern.jpg`.
